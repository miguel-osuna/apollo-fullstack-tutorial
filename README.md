## File structure

The app is split out into two folders:
- `client`: Client code
- `server`: Server code 

## Installation

To run the app, run these commands in two separate terminal windows from the root:

```bash
cd code/client && npm i && npm start
```

and

```
cd code/server && npm i && npm start 
```
